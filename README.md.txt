# SMA NetGuard

Protótipo de um Sistema Multiagente para Monitoramento de Redes desenvolvido como trabalho acadêmico.

## Funcionalidades

- Coleta de métricas de rede
- Análise de latência e tráfego
- Tomada de decisão baseada em eventos
- Notificação de alertas
- Dashboard Web para visualização em tempo real

## Tecnologias

- Python
- Flask
- HTML/CSS
- Arquitetura Multiagente
